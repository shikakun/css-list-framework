# CSS List Framework

- [Sass code](https://github.com/shikakun/css-list-framework/blob/master/src/styles/_list.scss)
- [Demo page](https://shikakun.github.com/css-list-framework/)

## CSS classes & HTML attributes

```html
<ul class="list">
  <li class="item">content</div>
  <li class="item">content</div>
</li>
```

- `.list`
    - `.-bullet` || `.-ordered`
    - `.-dense`
- `.item`
    - `.-icon` && `data-icon`
